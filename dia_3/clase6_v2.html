Clase 6: Visualización con Matplotlib

Graficar para comunicar

  • Matplotlib como herramienta básica
  • Un gráfico es un objeto de comunicación, no decoración

Motivación: ¿por qué graficar?

Un gráfico ayuda a responder rápido:

  • Tendencia: ¿sube o baja?
  • Comparación: ¿quién es mayor/menor?
  • Distribución: ¿hay muchos valores pequeños o grandes?

Note

Primero define la pregunta, luego el tipo de gráfico.

¿qué cuesta ver?

  • profundidad (m): [0, 50, 100, 150, 200]
  • temperatura (°C): [15.0, 18.2, 22.1, 26.0, 29.4]

Pregunta: sin graficar, ¿qué tan claro es el patrón?

Identificación de patrones

  1. ¿Aumenta o disminuye la temperatura con la profundidad?
  2. ¿Es “lineal” o cambia la pendiente?

Conclusión: la lista ayuda, pero el gráfico lo hace inmediato.

Setup: matplotlib como módulo

Matplotlib se usa típicamente con:

import matplotlib.pyplot as plt

pyplot (plt) ofrece funciones que dibujan: plot, scatter, bar, etc.

Regla práctica:

  • Primero: datos en listas/arreglos
  • Luego: una función de plt para dibujar
  • Finalmente: título/etiquetas para comunicar

Idea clave: el “mínimo” de un buen gráfico

Opcional: plt.grid(True) para lectura.

Comparación por categorías

Cuando la pregunta es:

  • “¿Cuántos hay en cada categoría?”

Un gráfico de barras suele ser adecuado.

Concepto: barras con plt.bar

Se necesitan:

  • categorías (texto)
  • valores (números)

Ejemplo típico:

  • modos de transporte elegidos en una encuesta
  • número de respuestas por opción

Ejercicio: construir conteos y graficar

Encuesta:

  • Cada elemento es el transporte principal: "bus", "metro", "auto", "bici", "a_pie"

Tarea:

  1. crea un diccionario conteos con el total por categoría usando un for
  2. grafica con plt.bar
  3. agrega título y ejes

Solución

Demo: dos curvas y una leyenda

Situación: reacción simple donde

  • reactivo disminuye
  • producto aumenta

Cuando hay más de una curva, necesitas:

  • label=...
  • plt.legend()
# Química: concentraciones vs tiempo
tiempo_s = np.linspace(0, 50, 101)

# Modelo simple: A -> B (cinética de primer orden)
k = 0.06
concentracion_A = 1.0 * np.exp(-k * tiempo_s)
concentracion_B = 1.0 - concentracion_A

Resultado

plt.plot(tiempo_s, concentracion_A, label="Reactivo A")
plt.plot(tiempo_s, concentracion_B, label="Producto B")

plt.title("Reacción A → B: concentración vs tiempo")
plt.xlabel("Tiempo (s)")
plt.ylabel("Concentración (u.a.)")
plt.legend(loc="lower left")

Subplots: varias vistas en una figura

A veces una sola gráfica no es suficiente.
Subplots permiten mostrar múltiples gráficos relacionados en una misma figura.

Idea clave

  • Una figura puede contener varios ejes
  • Cada eje es un gráfico independiente
  • Útil para comparar patrones con los mismos datos o mismas escalas

Esquema básico

fig, ejes = plt.subplots(1, 2)   # 1 fila, 2 columnas

ejes[0].plot(x, y1)
ejes[0].set_title("Gráfico A")

ejes[1].plot(x, y2)
ejes[1].set_title("Gráfico B")

plt.show()

Ejemplo

# Crear subplots: 1 fila, 2 columnas
fig, ejes = plt.subplots(1, 2)

# Subplot 1: concentración vs tiempo
ejes[0].plot(tiempo_s, concentracion_A)
ejes[0].set_title("Concentración A")
ejes[0].set_xlabel("Tiempo (s)")
ejes[0].set_ylabel("Concentración (u.a.)")

# Subplot 2: temperatura vs tiempo
ejes[1].plot(tiempo_s, concentracion_B)
ejes[1].set_title("Concentracion B")
ejes[1].set_xlabel("Tiempo (s)")
ejes[1].set_ylabel("Concentración (u.a.")
Text(0, 0.5, 'Concentración (u.a.')
